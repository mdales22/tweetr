public function getFriendsTimeLine(since_id:String = null, since_date:String = null, max_id:String = null, count:Number = 0, page:Number = 0):void

Returns the 20 most recent statuses posted by the authenticating user and that user's friends. This is the equivalent of /timeline/home on the Web. Note: Retweets will not appear in the friends_timeline for backwards compatibility. If you want retweets included use getHomeTimeLine.

This method requires Authentication

Parameters

since_id:String (default = null) — Optional. Returns only statuses with an ID greater than (that is, more recent than) the specified ID.
 
since_date:String (default = null) — Optional. Narrows the returned results to just those statuses created after the specified HTTP-formatted date, up to 24 hours old.
 
max_id:String (default = null) — Optional. Returns only statuses with an ID less than (that is, older than) the specified ID.
 
count:Number (default = 0) — Optional. Specifies the number of statuses to retrieve. May not be greater than 200.
 
page:Number (default = 0) — Optional. Provides paging. Ex. http://twitter.com/statuses/user_timeline.xml?page=3
